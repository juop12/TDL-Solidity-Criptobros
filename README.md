# React App setup

We use node v18.12.1. You can use a Node Version Manager (like nvm) for multiple node versions if needed.

- To install nvm & node:

[install nvm](https://github.com/nvm-sh/nvm). Then run at `front` folder:
```
nvm install 18.12.1
nvm use 18.12.1
```

- To start React App:

Run at `front` folder:
```
npm install
npm start
```
